OREKA plugin for WordPress helps you easily create a different search experience on your own WordPress website.


== Description ==
OREKA is an innovative solution for enterprise search that enables enterprises to use complicated search modules without any effort.
This version published with following feature:

* Easy to integrate and setup
* Automatic data ingestion
* Use your own search bar style
* Replace Wordpress default search with OREKA powerful search service with one click
* Optional auto-complete and auto-correct search query
* Intelligent query suggestion
* Sort widget
* Filter widget and search related facets
* Free plans for small businesses

It takes only two minutes to configure the plugin's graphical control panel once you register for free and install the plugin.
Contact us: [plugin@oreka.cloud](mailto://plugin@oreka.cloud)

== Instalation ==
* Add plugin to your Wordpress
* Use register tab to register free oreka search account
* Confirm your email address ( check spam folder )
* Enter the information sent in the email in the plugin settings tab
* Sync your wordpress data with oreka servers ( ingestion  tab )
* Start using our powerfull search service :)
